This HTML file is a workbench to prepare examples with <strike> and <u> tags. Then the examples are inserted into the Markdown files. The only problem, a big one, is that the Github renderer ignores <u> tags.

When using with Emacs and its HTML mode, you might want to use CTRL+TAB, to hide HTML tags and see only the bare text. This is a toggle command. Typing it a second time redisplays the HTML tags.

Another interesting function is CTRL-C CTRL-V, which displays the HTML file into your usual web browser. A variant is CTRL-C CTRL-S, which displays the HTML file into your usual browser each time it is saved.

A not interesting function is CTRL-C CTRL-T. You select a string with the mouse, then type CTRL-C CTRL-T and the name of an HTML tag. Emacs insert start / stop tags at both ends of the selected string. The problem is that at the same time, Emacs autoindents the HTML text and insert linefeeds at places where you do not want them.

Ce fichier HTML est un atelier pour préparer les exemples contenant des balises <strike> et <u>. Ensuite, on insère le source HTML directement dans le fichier Markdown. Le seul problème, et c'est un gros problème, est que le convertisseur Markdown → HTML de Github laisse tomber les balises <u>.

Si vous utilisez Emacs, sachez qu'il existe une fonction CTRL-TAB, qui masque les balises HTML et vous permet ainsi de visualiser le texte seul. Cette commande fonctionne en bascule. En l'utilisant une deuxième fois, les balises HTML réapparaissent.

Une autre fonction intéressante est CTRL-C CTRL-V, qui ouvre un onglet dans votre navigateur Internet habituel et y affiche le fichier HTML en cours d'édition. Une variante est CTRL-C CTRL-S, qui réaffiche le fichier dans le navigateur chaque fois que vous le sauvegardez.

Une fonction qui n'est pas intéressante est CTRL-C CTRL-T. Vous sélectionnez une partie de texte à la souris, vous tapez CTRL-C CTRL-T puis un nom de balise HTML. Emacs insère une balise de début et une balise de fin de part et d'autre du texte sélectionné. Hélas, Emacs en profite pour changer l'indentation de votre texte HTML et pour insérer des retours à la ligne à des endroits où vous n'avez pas envie. Tant pis, je n'utilise pas cette fonction.
